용도에 따라서 각 색상은 12가지 스케일로 분리됩니다.

- Scale 1: 페이지 배경
- Scale 2: 예비
- Scale 3: 컴포넌트 배경
- Scale 4: 컴포넌트 배경 (hover)
- Scale 5: 컴포넌트 배경 (active)
- Scale 6: 예비
- Scale 7: 경계
- Scale 8: 경계 (hover)
- Scale 9: 솔리드 배경
- Scale 10: 솔리드 배경 (hover)
- Scale 11: 글자 (low contrast)
- Scale 12: 글자 (high contrast)
